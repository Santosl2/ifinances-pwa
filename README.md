# <strong>Installation</strong>

1- Clone this repo

2- Run the command

```bash
yarn && yarn dev
```

or

```bash
npm i && npm run dev
```

<br/>

# Storybook

1- Just run the command

```bash
yarn storybook
```

```bash
npm run storybook
```
